Nashville Housing Analysis

Overview

This project focuses on analyzing real estate data from Nashville to identify undervalued properties and assist stakeholders in making data-driven investment decisions. By leveraging machine learning models and exploring key variables such as sale price and building value, the project highlights actionable insights for strategic portfolio optimization.

Dataset

The dataset includes over 22,000 property records from Nashville, encompassing 27 variables. Key variables include:

- Sale Price: The actual transaction price of the property.

- Building Value: The assessed value of the building.

- Land Value: The assessed value of the land.

- Finished Area: The total finished area in square feet.

- Property Type: The type of property (e.g., residential, commercial).

The dataset also categorized properties into three classes based on their valuation:

- Undervalued: Sale price is significantly lower than the combined building and land values.

- Overvalued: Sale price is significantly higher than the assessed value

Methodology

- Data Cleaning and Preparation:
Addressed missing values through imputation techniques.
Encoded categorical variables for model compatibility.
Handled outliers using the interquartile range (IQR) method.

- Exploratory Data Analysis:
Analyzed variable correlations and identified key predictors using a correlation heatmap.
Studied data distributions through histograms and boxplots.

- Model Implementation:
Developed Decision Tree, Random Forest, and XGBoost models.
Applied cross-validation to evaluate model performance and ensure generalizability.

- Evaluation Metrics:
Accuracy, precision, recall, and feature importance were used to compare models.

Key Findings

- Model Performance:
XGBoost: Achieved the highest accuracy at 98.9%, demonstrating robust prediction capabilities.
Random Forest: Performed well with an accuracy of 94.5%
Decision Tree: Achieved an accuracy of 88.6%, suitable for simpler scenarios.

- Top Predictors:
Land Value: Strongly correlated with undervalued properties.
Building Value: Highly indicative of property pricing.
Finished Area:Larger areas tend to be undervalued.

- Insights:
Properties with high land and building values but low sale prices are prime candidates for investment.
The model provides a reliable tool for distinguishing between overvalued and undervalued properties, reducing investment risks.
